9.5 증서발급
  
1) 증서발급 선택 시 새창이 나타남
2) ①신규부서명을 선택
3) ①문서종류를 선택
  - 부서 선택시 부서별 관련 문서종류가 나타나고 문서종류 선택시 ③편집창 내용이 변경
4) 발금통수, 요청자, 요청자 주민번호, 요청자 주소, 발급사유를 입력하고 *의 경우 필수 항목
5) 문서번호 발행을 선택하면 자동으로 문서번호가 등록
  - 문서선택시 문서번호가 등록되지 않습니다 라는 문구가 나타나 경우 홈페이지에서 출력되는 폼 확인용
6) ③편집창에서 작성 완료 후 저장 또는 인쇄
7) 메일 전송이 필요한경우 오른쪽 마우스 클릭하여 실행모드로 선택한 후 다시 마우스 오른쪽 이메일로 첨부
